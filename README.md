# Shipfall — Desktop Client

Shipfall is a lightweight gaming voice, video, and chat app. This repository hosts the **Windows
installer downloads** for testers. The source code is private and is **not** published here.

## Download
Get the latest installer from the **[Releases](../../releases)** page → `Shipfall-Setup-*.exe`.

## Install
1. Run the installer (installs per-user; no admin needed).
2. **Windows SmartScreen** may warn *"unknown publisher"* — this build isn't code-signed yet.
   Click **More info → Run anyway**.
3. Your **antivirus** may flag it as a false positive (the app can use a global push-to-talk key
   hook). If it quarantines the file, add an exception or restore it.
4. Open Shipfall and **register, then join with the invite code your host gave you.** The client is
   pre-configured to connect to your host's server — there's no address to type.

> Early test build — expect rough edges.
